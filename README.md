# json to go plugin for Inkdrop

A plugin to convert json to go structs

## Install

```shell
ipm install json-to-go
```

## Usage

    ```goStruct
    {
      "name":"Inkdrop",
      "Number": 1,
      "type": "App"
    }
    ```

It will be rendered as:
